# Network-Lab---Packet-Tracer

## Mô tả
Thực hành cấu hình hệ thống mạng gồm nhiều router và subnet trên Packet Tracer.
<img width="692" height="536" alt="image" src="https://github.com/user-attachments/assets/b4a2aa77-28f8-4d6c-b33f-d146464ae08d" />

## Nội dung thực hiện
- Cấu hình IP Addressing cho các router và thiết bị
- Thực hiện định tuyến giữa các mạng
- Thực hành ACL để kiểm soát truy cập
- Cấu hình Web Server và DNS Server trong mạng nội bộ
- Tìm hiểu cấu hình RADIUS theo lab

## Kiểm tra & Troubleshoot
- Kiểm tra kết nối giữa các thiết bị bằng ping
- Sử dụng Simulation để theo dõi đường đi của gói tin
- Kiểm tra trạng thái interface và cấu hình khi xảy ra lỗi

## Kết quả
- Các thiết bị trong mạng có thể kết nối với nhau
- Web Server và DNS hoạt động bình thường
- ACL hoạt động theo yêu cầu
